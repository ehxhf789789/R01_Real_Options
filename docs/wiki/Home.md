# BIM Real Options Valuation Model - Wiki

Welcome to the BIM Real Options Valuation Model wiki!

## About This Project

This project provides a Python-based valuation framework for BIM design service bidding decisions using Real Options Analysis (ROA). It helps construction companies make better bidding decisions by quantifying the strategic flexibility value that traditional NPV analysis misses.
